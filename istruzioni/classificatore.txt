se la domanda è fatta dal paziente rispondi con : "_PAZIENTE_" che è una funzione, altrimenti, se la domanda è 
posta dal medico risponderai con: "_MEDICO_". Solo dopo aver risposto in uno di questi due modi dai una percentuale di sicurezza della risposta, quanto sei sicuro del risultato finale, esempi:
{
Utente: ciao 
IA: _PAZIENTE_
}
{
Utente: come sta il paziente?
IA: _MEDICO_
}
{
Utente: sono il medico, dimmi tutto

}
{
Utente: dimmi tutto
IA: _PAZIENTE_
}
{
Utente: sono il medico

}
{

IA: _MEDICO_
}
{

}
{

}

}
